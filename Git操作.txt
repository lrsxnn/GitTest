克隆远程库
git clone git@github.com:lrsxnn/GtiTest.git
新建本地库
git init

查看本地变化
git status
对比文件改变
git diff <>
添加文件到暂存区
git add <>
提交文件
git commit -m ""
删除文件
git rm <>

查看版本历史
git log
查看分支合并图
git log --graph

查看分支
git branch
创建分支
git branch <>
切换当前分支
git checkout <>
创建并切换分支
git checkout -b <>
删除分支
git branch -d <>

查看远程库信息
git remote -v
推送分支
git push origin <>
关联远程库
git remote add origin git@github.com:lrsxnn/GtiTest.git
创建远程库的分支
git checkout -b <> origin/<>
建立本地分支与远程分支关联
git branch --set-upstream dev origin/dev
同步远程更改
git pull

查看标签
git tag
创建标签（指向HEAD)
git tag <>
指定标签信息
git tag -a <> -m ""
推送标签
git push origin <>
推送全部未推送过的标签
git push origin --tags
删除标签
git tag -d <>
删除远程标签
git push origin :refs/tags/<>

将指定分支与当前分支合并
git merge <>

放弃工作区的修改
git checkout -- <>
撤销缓存区修改
git reset HEAD <>
回滚上个版本
git reset --hard HEAD^
回滚上上个版本
git reset --hard HEAD^^
回滚上100个版本
git reset --hard HEAD^100
回滚指定版本
git reset --hard <id> 
查看命令历史以获取相应的版本ID
git reflog
